## Configuração do Repositório GitHub

Para colocar este projeto no GitHub, siga os seguintes passos:

1.  **Crie um novo repositório no GitHub:**
    *   Aceda a [github.com/new](https://github.com/new).
    *   Dê um nome ao repositório (ex: `Aulas_PA_25-26`).
    *   Não inicialize o repositório com README, .gitignore ou License (já temos esses arquivos localmente ou vamos criar).
    *   Clique em "Create repository".

2.  **Inicialize o repositório localmente (se ainda não o fez):**
    Abra o terminal na pasta do projeto (`/home/nunobifes/Aulas 25-26/Aulas_PA_25-26`) e execute:
    ```bash
    git init
    ```

3.  **Adicione os ficheiros e faça o commit inicial:**
    ```bash
    git add .
    git commit -m "Initial commit"
    ```

4.  **Ligue o repositório local ao GitHub:**
    Substitua `SEU_UTILIZADOR` pelo seu nome de utilizador do GitHub e `NOME_DO_REPO` pelo nome que deu ao repositório.
    ```bash
    git branch -M main
    git remote add origin https://github.com/SEU_UTILIZADOR/NOME_DO_REPO.git
    git push -u origin main
    ```
